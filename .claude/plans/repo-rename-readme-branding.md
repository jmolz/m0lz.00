# Feature: GitHub Repo Rename + README Catalog Branding

> Generated by `/plan-feature` — read CLAUDE.md and referenced files before implementing.

## Overview

Rename 3 GitHub repos to their m0lz catalog IDs and update each repo's README with a variant-specific branch mark SVG and catalog identity header. This completes the catalog identity system that the website already reflects.

## User Story

As a visitor following a project link from m0lz.dev, I want to land on a GitHub repo that carries the same catalog identity (branch mark, catalog ID, consistent naming), so that the brand feels coherent across the website and GitHub.

## Success Criteria

- [ ] 3 repos renamed: `pice-framework` → `m0lz.02`, `mcp-guard` → `m0lz.03`, `case-pilot` → `m0lz.04`
- [ ] Each repo has a `branch-mark.svg` file with the correct variant pattern
- [ ] Each repo's README has a centered branch mark + catalog ID header
- [ ] Old repo URLs auto-redirect to new names (GitHub handles this)
- [ ] No broken links or missing files in any repo

- **Complexity**: Low
- **Systems Affected**: 3 external GitHub repos (not m0lz.00)

### Patterns to Follow

**SVG base template (from `public/branch-mark.svg`):**

```svg
<svg width="48" height="48" viewBox="0 0 48 48" fill="none" xmlns="http://www.w3.org/2000/svg">
  <rect x="1.33" y="1.33" width="45.33" height="45.33" rx="5.33" fill="#000000" stroke="#333333" stroke-width="0.67"/>
  <line x1="18.24" y1="6.67" x2="18.24" y2="41.33" stroke="#ffffff" stroke-width="2.67" stroke-linecap="round"/>
  <line x1="29.76" y1="6.67" x2="29.76" y2="41.33" stroke="#ffffff" stroke-width="2.67" stroke-linecap="round"/>
  <!-- variant-specific branch lines go here -->
</svg>
```

**Branch coordinate math (48px):**

- Padding: `(5/36) * 48 = 6.67`
- LeftTrunkX: `48 * 0.38 = 18.24`
- RightTrunkX: `48 * 0.62 = 29.76`
- BranchLen: `(8/36) * 48 = 10.67`
- Y for fraction `f`: `6.67 + f * 34.67`
- Left branch endX: `18.24 - 10.67 = 7.57`
- Right branch endX: `29.76 + 10.67 = 40.43`

**README header pattern (from m0lz.00):**

```html
<p align="center">
  <img src="branch-mark.svg" width="48" height="48" alt="{catalogId} branch mark — {variant} variant">
</p>

<h1 align="center">{catalogId}</h1>

<p align="center">
  <strong>{Project Name}</strong> — {one-line description}<br>
  <a href="https://github.com/jmolz/{catalogId}">github.com/jmolz/{catalogId}</a>
</p>
```
