### Patterns to Follow

**Title format for catalog projects:**

```
# About page (the gold standard — already consolidated):
m0lz.02 — AI coding workflow orchestrator
m0lz.03 — MCP security proxy daemon
m0lz.04 — AI legal case management

# Blog post titles should follow the same pattern:
title: "m0lz.02 — Structured AI Coding Workflows"
title: "m0lz.03 — MCP Security Proxy"
title: "m0lz.04 — AI Legal Case Management"
```

**Name replacement rule (applies to ALL content — titles, headings, body text):**

Every reference to a project uses its catalog ID. No exceptions for body text or headings.

```
# Replace ALL standalone project references:
PICE           → m0lz.02        (when referring to the project)
PICE's         → m0lz.02's
MCP-Guard      → m0lz.03
MCP-Guard's    → m0lz.03's
Case Pilot     → m0lz.04

# KEEP these unchanged (actual CLI/package/protocol names):
pice plan, pice execute, pice evaluate    — CLI commands
mcp-guard init, mcp-guard               — CLI commands
@jacobmolz/mcpguard                     — npm package name
case-pilot/                             — directory names in code blocks
pnpm benchmark                          — build commands

# Methodology acronym expansion is OK on first mention:
"m0lz.02 (Plan → Implement → Contract-Evaluate) is a..."
# But do NOT use "PICE" as a standalone name.
```

**Heading format:**

```
# Old: ## How This Maps to PICE
# New: ## How This Maps to m0lz.02

# Old: ## WISC Context Management
# New: ## WISC Context Management  (WISC is a sub-concept, not a project name — keep as-is)

# Old: ## Self-Evolving Verification Frameworks: State of the Art and Blueprint for PICE
# New: ## Self-Evolving Verification Frameworks: State of the Art and Blueprint for m0lz.02
```

**Naming:** kebab-case files, PascalCase components, camelCase functions.

**Imports:**


## Notes

### Design Decisions

- **Old brand names fully replaced** — PICE, MCP-Guard, and Case Pilot are replaced with catalog IDs (m0lz.02, m0lz.03, m0lz.04) everywhere in content: titles, headings, body text, descriptions.
